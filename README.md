# chaos
Chaos is a simple 2D iterated function system plotter.

# Example
Create a file called *demo.ch* that contains:
```
4
    0.01 0.85 0.07 0.07 
4 3
    0.0 0.0 0.5
    0.85 0.64 0.075
    0.2 -0.26 0.4
    -0.15 0.28 0.575
4 3
    0.0 0.16 0.0
    -0.04 0.85 0.18
    0.23 0.22 0.045
    0.26 0.24 -0.086
```
Then open Chaos, click File->Open and open the previous file. <br/>
Drag a slider called *Speed* to 10.
Then you will see:
